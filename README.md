3D Particle System written with [OpenFrameworks](https://openframeworks.cc/)

Particles interact with the force point and the wind

Scene have: 
    Objects:
        Gift box
        Plane (planar mesh)
        Bonfire
        Christmas tree

    Force:
        Force point
        Wind
    
    Emitter:
        Snow
        Smoke/Fire ish from bonfire
